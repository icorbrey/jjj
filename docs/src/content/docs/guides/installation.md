`jjj` is currently in pre-alpha development, and you may encounter graphical or
logical bugs. If you find something wrong, please create a [new issue][issue]
on the GitHub repository.

Thanks for checking out `jjj`!
:::

## Cargo

The only installation method currently available is through `cargo`. You can
install `jjj` by running the following command in your terminal:

```sh
cargo install jjj
```

:::note[Don't have Cargo installed?]
Cargo is the package manager for the Rust programming language. The easiest way
to get it up and running is through [rustup](https://rustup.rs/).
:::
