## Frequently asked questions

### What's Jujutsu?

Jujutsu is a version control system that utilizes a more intuitive mental model
for change history and how we interact with it. It takes a lot of good ideas
from Git, Mercurial, and other legacy tools and merges them into the best
version control system seen to date. The best part is that Jujutsu is fully
compatible with Git, so you can use it today with forges like GitHub and Azure
DevOps!

<LinkButton icon="external" href="https://jj-vcs.github.io/jj/latest/">
  Learn more about Jujutsu VCS
</LinkButton>

### Why use Bevy?

Bevy is more than just a game engine! At its core, Bevy is a powerful and
beautifully ergonomic ECS (Entity-Component-System) framework that lets you
build applications of any kind. We use Bevy's data engine along with Ratatui to
build the interface of your dreams.
