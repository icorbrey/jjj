# `jjj`

A modal interface for [Jujutsu](https://jj-vcs.github.io/jj).

## ✨ Getting started

Clone the repository locally and build.

```sh
jj git clone https://github.com/icorbrey/jjj.git
cd jjj
cargo dev
```
